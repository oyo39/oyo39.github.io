CSS Flexbox từ Zero đến Pro

1. Giới thiệu Flexbox

Flexbox (Flexible Box) là module layout trong CSS giúp sắp xếp phần tử theo hàng hoặc cột, dễ xử lý alignment và phân phối khoảng trống trong container.

2. Thuộc tính của flex container

Cách biến một phần tử thành flex container:

.container {
  display: flex; /* hoặc inline-flex */
}

Các thuộc tính quan trọng:

  • flex-direction: row | row-reverse | column | column-reverse
  • flex-wrap: nowrap | wrap | wrap-reverse
  • justify-content: flex-start | center | space-between | space-around | space-evenly
  • align-items: stretch | flex-start | center | baseline | flex-end
  • align-content: khi có nhiều dòng

3. Thuộc tính của flex items

Các thuộc tính dùng trên phần tử con:

.item {
  order: 1;        /* vị trí hiển thị */
  flex: 1 1 auto;  /* grow | shrink | basis */
  align-self: center; /* override align-items */
}

4. Layout patterns

4.2 Header + content + sidebar

  display:flex;
  gap:16px;
}
.content { flex:1 }

4.3 Responsive card grid

  display:flex;
  flex-wrap:wrap;
  gap:12px;
}
.grid .card { flex:1 1 220px }

5. Ví dụ hoàn chỉnh

HTML:

<div class="center" style="height:200px">
  <div class="box">Nội dung</div>
</div>

CSS (gộp):

.center{display:flex;justify-content:center;align-items:center;height:200px;background:#f8fafc;border-radius:8px}
.box{padding:12px 18px;background:#60a5fa;color:#04263b;border-radius:6px;font-weight:700}
